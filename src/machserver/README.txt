This tree contains sources and other files for the Sprite on Mach
server.

directory		what it is
---------		----------
1.098			reference copy of the 1.098 native Sprite
			kernel (the Sprite server is based on this
			version)
include			header files: sources for user header files,
			symbolic links for kernel header files
lib			library binaries, lint library files
notes			random design notes, analysis results, etc.
paper			snippets from a note-yet-written paper
src			sources for the Sprite server, libraries, and
			user programs
tests			sources for test programs
tests.old		sources for out-of-date test programs
